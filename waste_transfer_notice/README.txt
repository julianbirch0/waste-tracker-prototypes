Waste Transfer Notice Prototype

This folder contains a fresh static browser prototype that recreates the government-style Duty of Care waste transfer note layout.

Source basis:
- The visual layout is based on the uploaded reference image of the WMC2A waste transfer note template.
- Footer wording uses: Based on WMC2A Version 3, August 2011.

Prototype mechanics:
- Pasted JSON input.
- A4 preview.
- Grid overlay and measurement readout.
- PDF download.
- JSON-controlled tick boxes.
- Sample JSON data for Sections A, B, C and D.

This is a first-pass recreation and will need visual tuning against the reference form.
