Waste Transfer Notice Producer to Carrier Prototype

This folder contains a static browser prototype for the Waste Transfer Notice Producer to Carrier document.

Initial scope:
- Pasted JSON input.
- Logo file 1 for the header.
- Logo file 2 for the footer.
- A4 preview.
- Grid overlay and measurement readout.
- PDF download.
- Footer box with Logo 2, strapline and website.
- Page number and document created timestamp.

This prototype intentionally starts with only the document title and footer mechanics. Document-specific fields and sections will be added incrementally.
